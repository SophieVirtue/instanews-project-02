# Instanews

This is a responsive web app that pulls articles from the New York Times API.

![](Giphy01.gif)

## Technologies Used

- HTML
- CSS/SASS
- JavaScript
- JQuery
- Ajax
- Gulp

## Installation

**Run locally with Gulp:**

```bash
npm install
```

**Build for production:**

```bash
npx gulp
```
